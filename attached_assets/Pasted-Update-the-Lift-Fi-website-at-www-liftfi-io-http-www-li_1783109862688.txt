Update the Lift Fi website at [www.liftfi.io](http://www.liftfi.io) for Lift Financial Holdings LLC.

The goal is to make the site look and feel like a serious, investor-ready private holding company: luxury, minimal, credible, structured, and compliant. The brand should feel closer to a private equity / holding company website than a startup landing page or crypto/stock app.

IMPORTANT:
Do not remove any important legal disclaimers.
Do not use language that makes Lift Financial Holdings sound like a bank, lender, investment advisor, broker-dealer, securities issuer, or guaranteed investment opportunity.
Use safe language such as “designed to,” “structured to,” “focused on,” “target structure,” “strategic growth,” “ownership interest,” and “private holding company.”

Main Tasks

1. Fix the broken scroll animations

The current fade-in / scroll reveal animation is causing sections to remain invisible or faded when scrolling quickly.

Fix the CSS and JavaScript so:

* Content is always visible if JavaScript fails.
* No section gets stuck at opacity: 0.
* Fade-in animations trigger reliably and smoothly.
* On mobile, animations should be subtle and not interfere with readability.

Preferred solution:

* Default content should be visible.
* Use a class like .js-enabled .fade-in only after JS loads.
* IntersectionObserver should add a visible class early enough, using a generous threshold/rootMargin.
* Respect prefers-reduced-motion.

If command-center.html exists, do not delete the file unless necessary. Instead:

* Remove public links to it.
* Add a note/comment in the code that this page is internal only.
* If the project supports routing/auth, protect it or make it non-public.
* If it cannot be protected, keep it unlinked and remove sensitive/internal checklist language from the public-facing experience.
